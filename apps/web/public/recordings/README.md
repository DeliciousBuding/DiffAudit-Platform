# Workspace Recordings

This directory is reserved for product recordings.

No recording binaries are committed in the public baseline. Regenerate release media from demo data and review it before publishing.

To refresh recordings, run the web app locally, sign in with a local test account, set `DIFFAUDIT_SESSION` to that local session value, and run the recording helper from `apps/web`.

Use a local demo account and avoid capturing environment-specific values.
