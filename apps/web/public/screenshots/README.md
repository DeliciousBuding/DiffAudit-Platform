# Workspace Screenshots

This directory is reserved for product screenshots used by docs or release material.

No screenshot binaries are committed in the public baseline. Regenerate screenshots from demo data and review them visually before committing.

To refresh screenshots, run the web app locally, sign in with a local test account, set `DIFFAUDIT_SESSION` to that local session value, and run the capture helper from `apps/web`.

Use a local demo account and avoid capturing environment-specific values.
